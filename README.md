# Filusha Bakes 🧁

Сайт домашней ремесленной кондитерской **Filusha Bakes** (Москва). Разработан на Next.js (App Router) и Tailwind CSS.

## Ключевые возможности

- **Каталог витрины:** Отдельные карточки, подробные составы с аллергенами, категории, поиск и сортировка.
- **Интерактивная корзина:** Мгновенный расчет стоимости, стильный drawer, поддержка промокодов (напр. `FILUSHA10`, `SWEET15`).
- **Оформление заказа:** Пошаговая форма, выбор интервалов доставки, симуляция (mock) онлайн-оплаты банковской картой.
- **Отслеживание заказа:** Проверка статуса по уникальному номеру вида `FILUSHA-YYYYMMDD-XXX` без логинов и паролей.
- **Админ-панель (`/admin`):** Управление статусами заказов с сохранением в `localStorage`.
- **Дизайн-система «Анти-ИИ»:** Насыщенная глубокая малиновая палитра (`#8B1A2B`), шрифты `Unbounded` и `IBM Plex Sans`, тактильные тени и плотный ремесленный стиль.

## Структура страниц

- `/` — Главная страница с витриной «Сейчас в витрине», фильтрами и формой быстрого заказа.
- `/catalog` — Каталог десертов с поиском и фильтром по категориям.
- `/category/[slug]` — Категории: Торты (`cakes`), Пирожные (`pastries`), Капкейки (`cupcakes`), Печенье (`cookies`).
- `/product/[slug]` — Страница товара с галереей, составом, аллергенами, степпером и кросс-сейлом.
- `/cart` — Страница корзины.
- `/checkout` — Оформление заказа и mock-оплата.
- `/checkout/success` — Подтверждение заказа с генерацией номера.
- `/order/[number]` — Публичный трекинг статуса заказа.
- `/admin` — Панель управления заказами.
- `/admin/order/[number]` — Изменение статуса отдельного заказа.
- `/about` — Страница с производственными стандартами и контактами.

## Запуск локально

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Деплой на Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

Проект полностью настроен для деплоя на Vercel:
- Хранение корзины и заказов через `localStorage` в клиентском контексте.
- Зафиксированные домены картинок в `next.config.ts`.
- Включены `.env.example`, `vercel.json` и `.gitignore`.
